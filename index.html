Programming Deeply Embedded Systems

I'm interested in Active Object systems and Hierarchical State Machines, and learned from Miro Samek, Phd.

I've written a Linux Shell and a RTOS kernel for ARM-CM4, but my main project lately has been a complete Active Object framework.

Some exciting features of my AO framework are dual targeted development (win and target hardware), a unique software tracing system with little overhead, as well as a unit testing system using Python!

I'm also interested in embedded application design using Hierarchical State Machines, the best known spagetti reducer. The user application is developed graphically, then is converted directly to C/C++ -- eliminating the need to maintain the graphical models and source code separately.
